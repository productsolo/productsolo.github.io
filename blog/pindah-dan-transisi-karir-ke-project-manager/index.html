Pindah dan Transisi Karir Ke Project Manager

24 Jul, 2024

oleh Mochammad Ihza Rizky Karim

Awalan

Menjadi Project Manager atau lebih tepatnya di artikel ini adalah IT Project Manager, sepertinya menarik jika berbicara mengenai project manager. Profesi IT Project Manager adalah sejatinya profesi yang berfungsi untuk memimpin, mengawal dan mengawasi proyek dari mulai skala kecil hingga besar, dari mulai sektor Teknologi Informasi hingga sektor manufaktur dan bangunan. Semua butuh IT Project Manager (baca: manajer proyek) karena profesi ini penting sekali untuk kelancaran proyek yang ada. Namun, ada 1 pertanyaan yang mengganjal, kalau bukan dari latar belakang project management apakah bisa menjadi IT Project Manager? jawabannya adalah bisa.

Lalu, Gimana Caranya?

Pertanyaan kedua, lalu bagaimana caranya menjadi IT project manager dengan tidak ada background sama sekali di IT Project Management?

Caranya adalah kamu bisa mengidentifikasi sendiri dari kesamaan background dan pengalaman kamu selama kuliah, berorganisasi dan magang. Lalu, kamu bisa cocokkan dengan daftar skill dibawah ini yang harus dimiliki oleh IT Project Manager:

1. Leadership Skill

Leadership skill adalah wajib dimiliki oleh setiap IT Project Manager / Project Manager. Menjadi Pemimpin Proyek berarti kamu mampu memimpin tim kamu dan stakeholder lainnya agar tim kamu bisa mengeksekusi proyek dengan baik dan tepat. Apakah kamu punya pengalaman memimpin tim dari skala kecil misalnya?…

2. Organizational Skill

Organizational Skill adalah keahlian dan kemampuan mengorganisir proyek dan flow dari tim kamu. Bagaimana kamu mengorganisir tugas-tugas mereka dan bagaimana kamu menginisiasi proyek, ini adalah penting dimiliki oleh setiap IT Project Manager / Project Manager. Apakah kamu punya organizational skill?…

3. Communication Skill

Komunikasi adalah hal wajib yang harus dikuasi jika kamu ingin transisi karir ke project management. Dengan komunikasi yang baik, percaya diri dan kuat, kamu bisa dengan mudah bernegosiasi dan memberikan arahan ke stakeholdermu nantinya.

4. Risk Management Skill

Manajemen Resiko adalah menjadi skill yang sangat penting jika kamu mau transisi dan pindah karir ke project manager. Skill ini jika kamu kuasai, akan memudahkan kamu dalam mengindentifikasi resiko yang ada dalam proyek dan bagaimana cara menanggulanginya kedepannyas selama proyek dijalankan.

5. Time Management Skill

Time Management Skill adalah kemampuan dan keahlian yang harus dikuasai oleh Project Manager. Karena kemampuan untuk mengelola waktu pribadi, tim, dan proyek adalah hal wajib.

6. Technical Skill

Technical skill atau keahlian teknis saat ini menjadi kewajiban untuk project manager ketahui seperti contohnya Project Manager wajib mengetahui IT jika proyek yang dia pegang merupakan proyek IT, atau misalkan Project Manager wajib paham bank bekerjanya seperti apa jika projectnya terkait perbankan dan seterusnya.

6. Paham beberapa metode project management

Paham metode project management dapat membantu kamu dalam mengorganisir task yang ada dan menjaga irama tim tetap on tha track dan selesai sesuai schedule. Metode Project Manangement yang bisa kamu pelajari seperti kanban, agile, scrum, waterfall, lean dan lainnya.

Tools, framework dan metode yang harus dipahami Project Manager:

  1. Jira
  2. Trello
  3. Slack
  4. Gantt Chart
  5. Google Workspace / Suite
  6. Microsoft Office
  7. Agile Scrum
  8. SDLC
  9. Scrum
  10. Waterfall
  11. Stakeholder Management
  12. The Eisenhower Matrix
  13. Git

Kesimpulan

Dengan memahami dan menjalan setidaknya 6 poin diatas, setidaknya kamu bisa mulai mentransisikan skill kamu sebelumnya dan menggabungkan dengan project management skill yang ada. Setelah itu, kamu bisa mulai membuat project kecil-kecilan dan menerapkan ilmu dan skillset sebagai junior project manager.
